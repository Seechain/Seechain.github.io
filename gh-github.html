GitHub CLI (gh)

gh es la CLI oficial de GitHub para gestionar repositorios desde la terminal.

Instalación

# Debian/Ubuntu
sudo apt install gh

# Fedora
sudo dnf install gh

# Arch
sudo pacman -S github-cli

Autenticación

gh auth login

Verificar estado:

gh auth status

Crear Repositorio

Crea un nuevo repositorio en GitHub:

gh repo create <nombre-repo>

Flags:

  • --source . - Usa el directorio actual como fuente
  • --public - Repositorio público
  • --private - Repositorio privado
  • --clone - Clona después de crear
  • --push - Hace push del código actual
  • --description "texto" - Descripción
  • --license "licencia" - Licencia (mit, gpl-3.0, etc)
  • --gitignore "plantilla" - Plantilla .gitignore

Ejemplos

Crear repositorio público desde directorio actual:

gh repo create mi-repo --source . --public

Crear repositorio privado y hacer push:

gh repo create mi-repo --source . --private --push

Crear repositorio vacío:

gh repo create mi-repo --private
