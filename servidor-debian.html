Debian 13 - Instalación Mínima

Instalación mínima de Debian 13 (Trixie)

Requisitos

  • USB bootable con Debian 13 netinstall
  • Conexión a internet

Proceso de Instalación

1. Descargar imagen ISO de debian.org

2. Crear USB bootable con dd o balenaEtcher

3. Boot desde USB

4. Seleccionar Install (no graphical install)

5. Configurar idioma, ubicación y teclado

6. Particionado: usar disco completo, todo en una partición

7. Selección de software:

- ✅ Servidor SSH

- ✅ Utilidades estándar del sistema

- ⬜ Entorno gráfico (NO seleccionado)

8. Completar instalación y reiniciar

Post-Instalación

# Actualizar sistema
sudo apt update && sudo apt upgrade -y

# Instalar herramientas básicas
sudo apt install curl wget git htop nano

Configuración de Red

# Ver configuración actual
ip addr

# Configurar IP estática (opcional)
sudo nano /etc/network/interfaces

Acceso SSH

# Desde otro equipo de la red
ssh usuario@ip-servidor
