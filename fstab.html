Montaje Automático con /etc/fstab

1. Identificar el disco a montar

# Ver todos los discos y particiones
lsblk -f

# Ver UUID específicos
sudo blkid

Ejemplo de salida:

sdb
└─sdb1 ext4  Backup   62147831-0f0d-4cc6-b820-854c51794513

2. Crear punto de montaje

# Crear directorio donde se montará el disco
sudo mkdir -p /mnt/backup

# Verificar
ls -ld /mnt/backup

3. Editar el archivo fstab

# Hacer backup
sudo cp /etc/fstab /etc/fstab.backup

# Editar fstab
sudo nano /etc/fstab

Agregar esta línea (según tu disco):

UUID=abcd1234-ab12-cd34-ef56-abcdefabcdef /mnt/backup ext4 defaults,noatime 0 2

Reemplaza el UUID por el de tu disco (ver paso 1). El UUID de arriba es solo un ejemplo.

Opciones comunes:

OpciónUso
defaultsOpciones estándar
noatimeMejor rendimiento
roSolo lectura
noautoMontaje manual

4. Probar la configuración

# Verificar sintaxis
sudo findmnt --verify

# Montar el disco
sudo mount -a

# Verificar montaje
df -h /mnt/backup

# Si hay error, ver logs
journalctl -xe | grep -i mount

Errores comunes:

ErrorSolución
mount point does not existsudo mkdir -p /mnt/backup
wrong fs typeVerificar tipo con lsblk -f
UUID=xxx does not existVer UUID con sudo blkid

¿Listo? Después de estos pasos, el disco se montará automáticamente en cada reinicio.
