Contraseñas seguras con openssl

¿Alguna vez has necesitado un token rápido o una contraseña segura y no querías usar una web externa? Tu terminal tiene la solución definitiva usando OpenSSL.

El Comando

openssl rand -base64 14 | tr -d '\n'

¿Por qué deberías usarlo?

  • Seguridad Criptográfica: A diferencia de otros comandos, openssl rand utiliza entropía del sistema para generar datos verdaderamente aleatorios.
  • Formato Base64: Al usar el flag -base64, transformamos bytes binarios en caracteres legibles (A-Z, a-z, 0-9, /, +), evitando caracteres de control que podrían dar problemas.
  • Salida Limpia: El comando tr -d '\n' elimina el salto de línea final. Esto es vital si estás copiando el token a un archivo .env o usándolo en un script, ya que evita espacios invisibles que suelen causar errores de autenticación.
