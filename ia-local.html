IA Local con Ollama

Guía de instalación de Ollama para tener modelos de IA funcionando de manera local en tu servidor.

Instalación de Ollama

En Ubuntu, la instalación es simple:

curl -fsSL https://ollama.com/install.sh | sh

Verificar que está funcionando:

ss -antp | grep :11434
curl http://localhost:11434 -v

Ollama queda disponible en http://127.0.0.1:11434 y se reinicia automáticamente con el sistema.

Gestión de Ollama

ollama --help - Muestra la ayuda general con todos los comandos disponibles.

ollama pull gemma3:1b - Descarga un modelo desde el repositorio de Ollama. Puedes ver los modelos disponibles en ollama.com/search.

ollama list - Lista todos los modelos descargados localmente.

ollama rm nombre-modelo - Elimina un modelo descargado locally.

ollama run gemma3:1b --verbose - Ejecuta un modelo en modo interactivo desde la terminal. El flag --verbose muestra estadísticas adicionales de la sesión.

Dentro de una sesión interactiva:

/? - Muestra la ayuda dentro de la sesión interactiva.

/show - Muestra información general del modelo.

/show info - Muestra detalles técnicos del modelo (parámetros, tamaño, etc).

/bye - Cierra la sesión interactiva con el modelo.

Acceso Externo

Por defecto Ollama solo acepta conexiones desde localhost. Para acceder desde otro equipo:

sudo systemctl edit ollama.service

Agregar:

[Service]
Environment="OLLAMA_HOST=0.0.0.0:11434"
Environment="OLLAMA_ORIGINS=*"

Recargar y reiniciar el servicio:

sudo systemctl daemon-reload
sudo systemctl restart ollama

Verificar que listens en todas las interfaces:

ss -antp

Debería mostrar:

LISTEN 0  4096  *:11434  *:*  users:(("ollama",pid=2101,fd=3))

Nota de seguridad: Abrir el acceso a red introduce riesgos. No hay autenticación por defecto. Restringe el acceso usando ACLs de red y nunca expongas Ollama directamente a internet.

Extensiones de Navegador

Puedes interactuar con Ollama directamente desde tu navegador usando extensiones.

Page Assist

Extensión de código abierto para Chrome, Firefox, Brave y Edge. Permite chatear con tus modelos locales desde cualquier página web.

  • Sidebar: Abre una barra lateral en cualquier página web
  • Web UI: Interfaz completa en una nueva pestaña
  • Chat With Webpage: Chatea sobre el contenido de la página actual
  • Atajos: Ctrl+Shift+Y (sidebar), Ctrl+Shift+L (Web UI)

Instalación:

  • Chrome Web Store
  • Firefox Add-ons
  • Edge Add-ons

Repositorio: github.com/n4ze3m/page-assist

Ollama Client

Extensión ligera con soporte multi-proveedor (Ollama, LM Studio, llama.cpp). Incluye RAG local y almacenamiento en IndexedDB.

  • Multi-proveedor: Conecta a Ollama, LM Studio o llama.cpp
  • Sidepanel: Chat integrado en el navegador
  • Local RAG: Workflows de recuperación de información local
  • Privacidad: Datos almacenados localmente, sin telemetría

Instalación:

  • Chrome Web Store
  • Firefox Add-ons

Repositorio: github.com/Shishir435/ollama-client
