Avahi - Configuración

Avahi permite resolver nombres .local en la red local sin necesidad de configurar un servidor DNS.

Instalación

sudo apt install avahi-daemon avahi-utils

Habilitar y arrancar

sudo systemctl enable avahi-daemon
sudo systemctl start avahi-daemon
sudo systemctl status avahi-daemon

Verificación

# Ver servicios publicados en la red
avahi-browse -a

# Resolver un hostname
avahi-resolve -n nombre.local

Configuración

El archivo principal está en /etc/avahi/avahi-daemon.conf

sudo nano /etc/avahi/avahi-daemon.conf

Opciones útiles:

[server]
use-ipv4=yes
use-ipv6=no

[publish]
publish-addresses=yes
publish-dns-servers=yes

Acceso al servidor

Después de configurar Avahi, puedes acceder al servidor usando:

ssh usuario@nombre-del-servidor.local

En lugar de recordar la dirección IP.
