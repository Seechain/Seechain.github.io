Termux - Configuración

Termux es un emulador de terminal para Android que permite ejecutar un entorno Linux nativo sin necesidad de root.

1. Activar el Almacenamiento

# Dar permisos de almacenamiento a Termux
termux-setup-storage

Explicación: Te pedirá permisos. Presiona "Permitir". Esto creará una carpeta ~/storage que enlaza a tu almacenamiento interno:

  • ~/storage/downloads → Carpeta Descargas
  • ~/storage/dcim → Fotos/Cámara
  • ~/storage/music → Música
  • ~/storage/shared → Raíz del almacenamiento

2. Actualizar Repositorios

# Actualizar lista de paquetes
pkg update

# Actualizar paquetes instalados
pkg upgrade -y

3. Instalar los Programas

# Instalar todo de una vez
pkg install -y mpv yt-dlp aria2 htop wget fastfetch yazi

Explicación de cada programa:

ProgramaFunción
mpvReproductor multimedia (video/audio)
yt-dlpDescargar videos/audio de YouTube y otros sitios
aria2Descargador multiprotocolo (HTTP/BitTorrent/Metalink)
htopMonitor de sistema (procesos, RAM, CPU)
wgetDescargar archivos desde internet
fastfetchMuestra información del sistema (alternativa moderna a neofetch)
yaziExplorador de archivos desde terminal

4. Instalar y Configurar la Fuente (Hack Nerd Font)

# 1. Crear carpeta de configuración de Termux
mkdir -p ~/.termux

# 2. Moverse a la carpeta de configuración
cd ~/.termux

# 3. Descargar la fuente Hack Nerd Font
wget https://github.com/ryanoasis/nerd-fonts/raw/master/patched-fonts/Hack/Regular/HackNerdFont-Regular.ttf

# 4. Renombrar la fuente
mv HackNerdFont-Regular.ttf font.ttf

# 4. Recargar la configuración de Termux
termux-reload-settings

5. Termux-API (Acceso al Hardware del Teléfono)

Permite acceder a sensores y funciones del teléfono desde la terminal.

# Instalar el paquete
pkg install termux-api

# Ejemplos de uso
termux-battery-status       # Estado de la batería
termux-camera-photo         # Tomar foto
termux-clipboard-get        # Obtener texto del portapapeles
termux-clipboard-set        # Establecer texto en el portapapeles
termux-notification         # Enviar una notificación
termux-wifi-scaninfo        # Escanear redes WiFi
termux-sensor -s            # Listar sensores disponibles
termux-microphone-record    # Grabar audio

Requisito adicional: Instalar la app Termux:API desde F-Droid.

6. SSH desde Termux

Usar OpenSSH para conectarte a servidores o iniciar un servidor SSH en tu dispositivo.

# Instalar OpenSSH
pkg install openssh

# Conectarse a un servidor remoto
ssh usuario@servidor

# Generar clave SSH
ssh-keygen -t ed25519

# Iniciar servidor SSH en Termux (puerto 8022)
sshd -p 8022

# Ver tu IP local
ip a
