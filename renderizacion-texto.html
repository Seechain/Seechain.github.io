Renderización mediante Texto

En el desarrollo web y la administración de sistemas, la eficiencia y la estética convergen en el uso de formatos basados en texto. Este enfoque reduce peticiones HTTP y permite integración profunda con herramientas de terminal.

2. Nerd Fonts

Nerd Fonts es un proyecto que parcha (modifica) fuentes populares de programación añadiendo miles de iconos, glifos y símbolos.

@import "https://www.nerdfonts.com/assets/css/webfont.css";

Uso: <i class="nf nf-fa-heart"></i>

Instalación en Linux

# Descargar instalador
curl -L https://github.com/ryanoasis/nerd-fonts/releases/download/v3.0.2/FiraCode.zip -o FiraCode.zip
unzip FiraCode.zip -d ~/.local/share/fonts/FiraCode
fc-cache -fv

Instalación en Windows

# Via Scoop
scoop install nerd-fonts

# O manualmente en %WINDIR%\Fonts

3. SVG

SVG (Scalable Vector Graphics) es un formato de imagen vectorial basado en lenguaje XML, diseñado específicamente para mostrar gráficos bidimensionales, logotipos e ilustraciones en la web.

<svg height="280" width="360" xmlns="http://www.w3.org/2000/svg">  
  <polygon points="150,15 258,77 258,202 150,265 42,202 42,77"  
  style="fill:lime;stroke:purple;stroke-width:3" />  
</svg>

4. Base64: Binarios en texto

Base64 es un sistema de codificación que transforma datos binarios (imágenes, archivos, audio) en una cadena de caracteres ASCII seguros (texto imprimible). Permite transmitir información binaria a través de protocolos diseñados para texto, como correos electrónicos (MIME) o JSON.

<img src="data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAAAUA
    AAAFCAYAAACNbyblAAAAHElEQVQI12P4//8/w38GIAXDIBKE0DHxgljNBAAO
    9TXL0Y4OHwAAAABJRU5ErkJggg==" alt="ejemplo" />

Linux

# Codificar imagen
base64 -w 0 imagen.jpg > imagen.txt

# Desde URL
curl -s https://ejemplo.com/imagen.jpg | base64 -w 0

Windows (PowerShell)

[Convert]::ToBase64String((Get-Content -Path "imagen.jpg" -AsByteStream)) | Set-Content -Path "imagen.txt"
