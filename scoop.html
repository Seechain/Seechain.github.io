Scoop - Gestor de Paquetes para Windows

Scoop es un gestor de paquetes desde línea de comandos para Windows que simplifica el proceso de instalar, actualizar y gestionar aplicaciones. Es el equivalente de Windows a Homebrew o apt-get.

Scoop descarga y gestiona paquetes de forma portátil, lo que significa que no se requieren permisos de administrador para instalar nada. Una vez instalado, Scoop coloca las aplicaciones en ~\scoop.

Instalación

Para instalar Scoop, ejecuta el siguiente comando en PowerShell:

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
Invoke-RestMethod -Uri https://get.scoop.sh | Invoke-Expression

Comandos Básicos

Buscar un paquetescoop search [nombre_paquete]
Instalar un paquetescoop install [nombre_paquete]
Actualizar un paquetescoop update [nombre_paquete]
Listar paquetes instaladosscoop list
Desinstalar un paquetescoop uninstall [nombre_paquete]

Oh My Posh

Oh My Posh es un motor de temas para el prompt de PowerShell que permite personalizar el terminal. Es la versión para Windows de la popular herramienta Oh My Zsh usada en sistemas Unix.

Instalación

Para instalar Oh My Posh usando Scoop, ejecuta:

scoop install main/oh-my-posh

Configuración

Para configurar un tema, primero necesitas inicializar Oh My Posh en tu perfil de PowerShell. Abre el perfil ejecutando:

notepad $PSHOME\Profile.ps1

Añade la siguiente línea al script del perfil:

oh-my-posh init pwsh --config "$env:POSH_THEMES_PATH\tu-tema.omp.json" | Invoke-Expression

Reemplaza tu-tema.omp.json con el path al tema deseado. Los temas se encuentran en la carpeta themes dentro del directorio de instalación de Oh My Posh.

YT-DLP

YT-DLP es un gestor de descargas de audio/vídeo desde línea de comandos con soporte para miles de sitios.

Instalación

scoop install yt-dlp ffmpeg

Uso

Descargar vídeo en MP4

yt-dlp -f "bestvideo[ext=mp4]+bestaudio[ext=m4a]/mp4" https://www.youtube.com/watch?v=ejemplo

Descargar MP3 con máxima calidad y metadatos

yt-dlp -x --audio-format mp3 --audio-quality 0 --embed-metadata --embed-thumbnail --add-metadata --extractor-args "youtube:player_client=android" -o "%(artist,creator,uploader)s - %(title)s.%(ext)s" https://www.youtube.com/watch?v=ejemplo

MPV

MPV es un reproductor de medios gratuito para línea de comandos.

Instalación

scoop install mpv

Uso

Reproducir un archivo de vídeo local:

mpv video.mp4

Reproducir desde una URL (streaming):

mpv https://www.twitch.tv/usuario
mpv https://www.youtube.com/watch?v=ejemplo

Reproducir solo audio (sin vídeo):

mpv --no-video https://www.twitch.tv/usuario
mpv --no-video https://www.youtube.com/watch?v=ejemplo

Bucle infinito (para música o streams):

mpv --no-video --loop https://www.youtube.com/watch?v=ejemplo
