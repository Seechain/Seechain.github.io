Fastfetch - Configuración

Fastfetch es una herramienta para mostrar información del sistema, similar a neofetch pero más rápida.

Nerd Fonts

Los iconos que ves en la configuración son símbolos de Nerd Fonts, una fuente que incluye miles de glyphs adicionales.

Cómo funcionan:

  • Los iconos como 󰌢, ,  son caracteres Unicode privados
  • Necesitas una fuente con parches Nerd Font en tu terminal
  • Ejemplos: JetBrainsMono Nerd Font, FiraCode Nerd Font, Hack Nerd Font

Instalación en Debian:

sudo apt install fonts-jetbrains-mono

Página oficial: nerdfonts.com

Logo personalizado

Ubicación: ~/.config/fastfetch/logo/logo.txt

Arte ASCII que se muestra junto a los módulos de información.
