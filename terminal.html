Comandos Terminal

Todos los comandos mostrados a continuación están diseñados para ejecutarse dentro de una Terminal.

Nota: Estos comandos están adaptados para Arch Linux y derivados (Manjaro, EndeavourOS, etc.). Para distribuciones basadas en Debian/Ubuntu, usa sudo apt install paquete.

YT-DLP

YT-DLP es un descargador de audio/video rico en funciones con soporte para miles de sitios.

Instala las herramientas necesarias para descargar audio y reproducir streams:

sudo pacman -S yt-dlp mpv ffmpeg

Para descargar un video de YouTube como archivo MP4:

yt-dlp -f "bestvideo[ext=mp4]+bestaudio[ext=m4a]/mp4" https://www.youtube.com/watch?v=ejemplo

Para descargar audio de un video de YouTube en formato MP3:

yt-dlp -x --audio-format mp3 https://www.youtube.com/watch?v=ejemplo

Para descargar audio de un video de YouTube en formato MP3 con la mejor calidad y metadatos incrustados:

yt-dlp -x --audio-format mp3 --audio-quality 0 --embed-metadata --embed-thumbnail --add-metadata --extractor-args "youtube:player_client=android" -o "%(artist,creator,uploader)s - %(title)s.%(ext)s" https://www.youtube.com/watch?v=ejemplo

MPV

MPV es un reproductor de medios gratuito para la línea de comandos.

Para ver un stream de Twitch con MPV:

mpv https://www.twitch.tv/nombre_usuario

Para escuchar solo el audio de un stream de Twitch:

mpv --no-video https://www.twitch.tv/nombre_usuario

Para reproducir un track de YouTube en bucle con solo audio:

mpv --no-video --loop https://www.youtube.com/watch?v=ejemplo

Para detener la reproducción en MPV, presiona Ctrl + C.

FFmpeg

FFmpeg es una herramienta poderosa de línea de comandos para procesar video.

Para capturar la pantalla del escritorio como archivo de video:

ffmpeg -f x11grab -framerate 30 -i :0.0 output.mp4

Este comando captura el escritorio a 30 FPS y lo guarda como output.mp4.

Presiona Q para detener la captura.

Para convertir un archivo de video a GIF:

ffmpeg -i output.mp4 -vf "fps=15,scale=600:-1:flags=lanczos" output.gif

Este comando toma output.mp4 como entrada, establece la velocidad de cuadros en 15 FPS, cambia la escala del video a un ancho de 600 píxeles mientras preserva la relación de aspecto, y usa el algoritmo de escalado Lanczos para mejor calidad.

Para generar imágenes en miniatura escaladas:

ffmpeg -i %d.jpg -vf "scale=320:-1" %d_thumb.jpg

Este comando lee una secuencia de imágenes llamadas 1.jpg, 2.jpg, 3.jpg y crea miniaturas escaladas con un ancho de 320 píxeles mientras preserva la relación de aspecto.

Fastfetch

Fastfetch es una herramienta ligera de información del sistema para la terminal.

Para instalar Fastfetch:

sudo pacman -S fastfetch

Para mostrar información del sistema, simplemente ejecuta:

fastfetch

Btop

Btop es un monitor de recursos moderno para la terminal.

Para instalar btop:

sudo pacman -S btop

Para iniciar el monitor de recursos:

btop

Aria2

Aria2 es un gestor de descargas ligero y rápido para línea de comandos. Soporta HTTP, HTTPS, FTP, BitTorrent y Metalink, y puede descargar múltiples archivos simultáneamente.

Para instalar Aria2:

sudo pacman -S aria2

Para descargar un archivo usando una URL directa:

aria2c https://ejemplo.com/archivo.zip

Para reanudar una descarga interrumpida:

aria2c -c https://ejemplo.com/archivo.zip

Chafa

Chafa es una utilidad de gráficos de terminal que renderiza imágenes usando caracteres y colores ANSI. Permite ver imágenes directamente en la terminal.

Para instalar chafa:

sudo pacman -S chafa

Para mostrar una imagen en la terminal:

chafa imagen.png

Chafa soporta múltiples modos de salida y se adapta automáticamente al tamaño y capacidades de color de la terminal.

Nano

Nano es un editor de texto simple y fácil de usar basado en terminal.

Para instalar nano:

sudo pacman -S nano

Para abrir o crear un archivo con nano:

nano nombre_archivo.txt

Atajos de teclado comunes en nano:

  • Ctrl + O — guardar el archivo
  • Ctrl + X — salir del editor
  • Ctrl + K — cortar la línea actual
  • Ctrl + U — pegar el texto cortado

FZF

FZF es un buscador fuzzy de línea de comandos que te permite buscar y seleccionar rápidamente archivos, directorios, procesos y más en tu terminal.

Para instalar fzf:

sudo pacman -S fzf

Puedes combinar fzf y nano para buscar rápidamente un archivo y editarlo en nano.

nano $(fzf)

Webtorrent-cli

Webtorrent-cli es un cliente torrent simple para usar en Node.js.

Primero, instala Node.js:

sudo pacman -S nodejs npm

Una vez que Node.js esté listo, instala Webtorrent-cli globalmente vía npm:

npm install webtorrent-cli -g

Uso

Descarga un torrent desde un archivo .torrent o un enlace magnet:

webtorrent "archivo.torrent"
webtorrent "magnet:?xt=urn:btih:..."

Streaming de video con MPV

Transmite el video directamente a MPV mientras se descarga, sin esperar a que el archivo se complete:

webtorrent "archivo.torrent" --mpv

Ani-cli

Ani-cli es una herramienta de línea de comandos para buscar anime directamente desde la terminal, usando reproductores externos como MPV y VLC para la reproducción.

Instalación

Instala las dependencias requeridas:

sudo pacman -S git fzf ffmpeg mpv python

Clona el repositorio y configura:

git clone https://github.com/pystardust/ani-cli /tmp/ani-cli
sudo cp /tmp/ani-cli/ani-cli /usr/local/bin/ani-cli
sudo chmod +x /usr/local/bin/ani-cli

Para soporte de descargas, también se recomienda instalar:

sudo pacman -S yt-dlp aria2

Uso

Buscar y ver un anime:

ani-cli one piece

Actualizar ani-cli:

ani-cli -U

Mostrar ayuda:

ani-cli -h
